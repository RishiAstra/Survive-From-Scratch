Animation info

Use Animator Override Controllers for characters' animations
character defined as Players, NPCs, Enemies, etc.

Idle-0 is used in movement blend trees. 
If the character is moving slowly, Idle-0 will be used with a high weight. At high speeds, Idle-0 is used with a low weight, blended with movement animations.

Set Attack trigger and AttackIndex to attack
Set IdleEnd to exit idle immediately. 
Set Idle trigger and IdleIndex to idle
