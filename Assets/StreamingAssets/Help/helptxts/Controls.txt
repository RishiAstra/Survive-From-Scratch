Use WASD to move
Space to jump
Mouse to look

Click to attack using the default attack skill. Note that when holding some items, such as slime goo, you cannot deal damage.
Hold alt, select with the mouse, and release alt to use a skill
Right-click to throw a rock (must be holding a rock)
F to pick up an item or interact with an NPC
Q to drop an item
E to show/hide crafting
M to show/hide map

1-9 to select that inventory slot

Esc to release the mouse

You can press the button in the upper right corner to enter/exit fullscreen.
